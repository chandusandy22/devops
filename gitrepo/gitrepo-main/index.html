GIT RESET COMMNAD

There are 3 Types of Reset commands in git:

  • Hard Reset
  • Soft Reset
  • Mixed Reset

Hard Reset

It resets the index and working tree in git. Any canges to track files in the woking tree since the commit are discarded.

$git reset --hard Previous-commitID

Soft Reset

This moves HEAD back to the specific commit, undoes all the changes made between where HEAD was pointing and the specific commit, and saves all the changes in the index.

$git reset --soft previous-commitid

Mixed Reset

This option will undo all the changes between HEAD and the specific commit. But will preserve changes in the working directory as unstaged changes.

$git reset --mixed previous-commitid
